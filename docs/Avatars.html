Avatars

Avatars are images that users can set as their profile picture.They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Round Avatars

The Avatars are available in 3 sizes as shown below which can be toggled by using --small, --medium and --large.

Small Avatar
Medium Avatar
Large Avatar

Implementation

Choose your Avatar size and off you go!

Squircle Avatars

The Squircle Avatars are available in 3 sizes as shown below which can be toggled by using --squircle along with the respective sizes using --small, --medium and --large.
